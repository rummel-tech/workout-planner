## [1.0.0] - 2025-01-01

### Added
- Weekly workout planning with 7-day calendar view
- Maximum 3 workouts per day enforcement
- Workout detail editor with warmup, main set, and cooldown sections
- Structured exercise tracking (sets, reps, weight, duration, distance, rest)
- Goal creation and workout-goal association
- User authentication (email/password and Google OAuth)
- Dashboard with daily overview and readiness score
- Profile and settings screens
- Apple HealthKit integration (iOS/macOS)
- AI Coach chat interface
- Multiple workout types support (Strength, Run, Swim, Murph, Bike, Yoga, Cardio, Mobility, Rest)
